# Group Component
Data about a top-level group and sub-groups.

- View
  - Group info - top-level info. Name, slug, logo, etc. 
  - Admins - Who can manage the group. A list of users.
  - Groups - Sub-groups within this group (hierarchical). 
  - Usage - stats for reports and pricing.
  - Titles - Group titles (leadership roles). Separate, but related to roles. 
  - Roles - List of roles and associated permissions Group users can have. 
- Edit - Update any group info. 
- Delete - Remove group. Needs protections. 

Namegame soft deletes groups to ensure one person can't destroy the group. 

Some of the group info is publically visible.
